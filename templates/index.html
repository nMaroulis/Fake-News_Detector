Fake News Detector

The following implementation was developed for the purposes of the Big Data Class Project (MSc.).

Use this website to analyze an article you found online, in order to find out if it's fake
The article classification is achieved using Deep Neural Nets in Tensorflow API
as well as other ML techniques like SVM, Decision Trees etc.
Over 60.000 articles were used for training provided by kaggle.com as well as the onion subreddit.
